# Experiment Docs

This directory documents the active experiment families.

- `experiment_overview.md` is generated from `configs/experiments/`.
- `umaxformer_v1_family.md` describes the SODA plus ORAS5 validation baseline.
- `umaxformer_v2_family.md` describes the promoted ORAS5-only family.
- `model_design_order_sheet.md` records the active model ordering.
