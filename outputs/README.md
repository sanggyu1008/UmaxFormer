# Outputs

Generated artifacts are organized by purpose:

- `experiments/cnn_transformer_v1/`
- `experiments/cnn_transformer_v2/`
- `ensembles/`
- `analyses/`
- `legacy_flat/`

New runs should write below `outputs/experiments/<experiment>/<run_name>`.
Legacy generated artifacts from the previous flat output layout are preserved under `legacy_flat/`.
